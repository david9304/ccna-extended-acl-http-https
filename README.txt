In this lab, I implemented a small network consisting of a web server, two routers, one switch, one wireless access point, one laptop, and two PCs.

The main objective was to restrict HTTP and HTTPS access to the web server using extended ACLs.

The network uses static routing on both routers to enable communication between the different LAN segments.

Router2 also functions as a DHCP server, providing IPv4 addresses and default gateway information for the two PCs.

The laptop is configured with a static IP address for testing purposes and to ensure consistent ACL behavior during the lab.
